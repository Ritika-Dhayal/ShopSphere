## Overview
ShopSphere is a pixel-perfect, static front-end clone of Amazon's homepage, recreated entirely using HTML5 and CSS3. It mimics the look and feel of a modern e-commerce platform — from the navigation bar and hero banner to product category cards and a detailed footer.

## Features
1.**Sticky Navigation Bar** — Amazon-style top bar with logo, search box, delivery location, account options, and cart
2.**Hero Banner Section** — Full-width promotional banner with text overlay and product imagery
3.**Product Category Cards** — Grid layout showcasing categories: Clothes, Health & Personal Care, Furniture, Electronic Devices, and more
4.**"See More" Links** — Category cards with call-to-action links
5.**Location-aware Banner** — Notification bar prompting users to switch to a regional store
6.**Footer** — Multi-column footer with sections: Get to Know Us, Make Money with Us, Amazon Payment Products, and Let Us Help You
7.**Back to Top Button** — Smooth scroll back to page top
8.**Fully Responsive Layout** — Adapts to different screen widths
